# WebShare

Moderní prezentační web postavený na Vite, React a Tailwind CSS.

## Stack
- Vite
- React
- React Router
- Tailwind CSS
- Framer Motion
- Lucide React

## Spuštění lokálně

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Struktura projektu
- `src/pages` – stránky
- `src/components` – znovupoužitelné komponenty a sekce
- `src/content` – obsah webu
- `src/i18n` – překlady a jazyková vrstva
- `src/hooks` – vlastní hooky
- `src/lib` – utility
